# Sai Harshitha — Portfolio

A cinematic, highly interactive personal portfolio built for performance, aesthetics, and dynamic presence. This portfolio showcases my journey as a Full-Stack Developer, Video Editor, and Powerlifter.

## 🚀 Live Demo
[View Live Site](https://my-portfolio-core-ctrl.vercel.app/) *(Replace with actual URL if different)*

## 🛠️ Tech Stack & Arsenal

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Library**: [React 18](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations & Physics**: [Framer Motion](https://www.framer.com/motion/)
- **Live Discord Presence**: [Lanyard API](https://github.com/Phineas/lanyard) (Real-time Spotify & VS Code coding status)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## ✨ Key Features

- **Cinematic Typography & Layout**: Huge dynamic text sizes utilizing CSS clamp and responsive design principles to create a premium, editorial feel.
- **Dynamic Meta Badges**: A top-left badge that continuously cycles through my current roles (Powerlifter, Developer, Creator) using React state and Framer Motion.
- **Real-Time Activity (Lanyard)**: A digital pixel cat companion on the screen that syncs with my live Discord status. When I'm coding in VS Code or Cursor, or listening to Spotify, the cat detects it and reveals a beautiful animated activity card!
- **Smooth Parallax Scrolling**: Complex scroll-linked animations and ambient mouse-tracking glow effects that respond to cursor movement.
- **Responsive & Optimized**: Fully optimized for mobile and desktop viewing with dynamic image and video rendering.

## 📦 Local Development

1. Clone the repository:
```bash
git clone https://github.com/core-ctrl/MY-Portfolio.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
